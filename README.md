# Oasis Real Estate Simulation (Dongguan Edition)

This is an agent-based model simulation of the Dongguan real estate market, featuring LLM-driven agents (Buyer/Seller), complex transaction logic, and market reporting.

### Core Components
- **`simulation_runner.py`**: The **Main Entry Point**. Run this to start the complete simulation.
- `generate_simulation_report.py`: Automatically generates reports (`agent_personas`, `decisions`, `market_report`, `negotiations`) after a run.
- `models.py`: Data models (`Agent`, `Market`, `Property`, etc.).
- `agent_behavior.py`: Core logic for Agent thinking, profiling, and decision making (LLM integration).
- `transaction_engine.py`: Handles listings, matching, negotiation, and transaction execution.
- `property_initializer.py`: Initializes the market with Dongguan-based property data (Zones A/B/C).
- `mortgage_system.py`: Loan and affordability calculations.

### Directories
- **`results/`**: Stores simulation output (CSVs and Logs) for each run, timestamped.
    - `agents.csv`, `properties.csv`, `trans.csv`, `thoughts.csv`, `console_log.txt`.
- **`reports/`**: Stores the latest analysis reports (Markdown).
